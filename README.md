# 📊 Wireshark Statistics and Graph Analysis

## 📌 Overview

This project demonstrates network traffic analysis using Wireshark and Python (Scapy). It includes packet capture, graphical visualization, and statistical analysis of protocols and conversations.

---

## 🛠️ Tools Used

* Wireshark
* Python
* Scapy

---

## 📊 Features

* 📈 IO Graph analysis (TCP, UDP, DNS, HTTP)
* 📡 Protocol hierarchy analysis
* 🔗 Conversation analysis
* 📦 Packet statistics computation

---

## 📂 Files Included

* `exp7capture.pcapng` – Captured network traffic
* `protocol_analysis.py` – Protocol statistics analysis
* `conversation_analysis.py` – Communication pair analysis
* Report document
* Graph screenshots

---

## ▶️ How to Run

```bash
pip3 install scapy
python3 protocol_analysis.py
python3 conversation_analysis.py
```

---

## 📈 Output

* Total number of packets
* Data size and header size
* Communication pairs with maximum data
* Inter-packet timing analysis

---

## 📌 Conclusion

This project provides insights into real-time network behavior using Wireshark’s statistical tools and Python-based packet analysis.
